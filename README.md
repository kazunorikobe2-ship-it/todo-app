# Kanban Board (todo-app)

Trello風のシンプルなカンバンボードアプリ。プレーンなHTML/CSS/JavaScriptのみで作られており、ビルド不要。

## 機能

- リスト(列)の追加・削除・リネーム
- カードの追加・編集(ダブルクリック)・削除
- ドラッグ&ドロップでカードをリスト間移動
- `localStorage` にブラウザ内で自動保存

## ローカルで開く

`index.html` をブラウザで開くだけで動作する。

## Vercelへのデプロイ

ビルド設定不要の静的サイトなので、Vercelでリポジトリをインポートするだけでデプロイできる。
